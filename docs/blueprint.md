# **App Name**: Revopz Power Hub

## Core Features:

- Informational Content Display: Render key informational pages including Home, About Us, Contact, and Warranty, with all specified sections and content, maintaining a clean and product-focused layout.
- Comprehensive Product Catalog: Enable users to browse lithium inverters, batteries, and all-in-one systems through category-based listings, with detailed product pages featuring images, specifications, and use cases.
- Multi-Channel Enquiry System: Provide a streamlined process for customer inquiries via direct WhatsApp integration, phone call initiation, and an in-app email submission form for product-specific, general, or dealership queries.
- Responsive Design System: Ensure a fully adaptive user interface that delivers an optimal viewing experience across desktop, tablet, and mobile devices, featuring large, tappable buttons for accessibility.
- Sticky Navigation Bar: Implement a persistent top navigation bar with quick access to main sections (Products, Technology, About, Contact, Warranty) and subtle hover effects for enhanced user interaction.
- Product Description Simplification Tool: An internal generative AI tool to assist content managers in transforming complex technical product details and features into easy-to-understand, jargon-free explanations suitable for middle-class customers and small businesses.

## Style Guidelines:

- Color scheme: Dark theme, featuring a black background and dark gray section dividers for a premium and technological feel.
- Primary color: An energetic electric blue (#1496E6) to represent technology and innovation, offering strong contrast against the dark background.
- Background color: A very dark desaturated blue-gray (#151B1D), providing depth while maintaining visual continuity with the primary electric blue.
- Accent color: A vibrant orange (#FF8000) reserved for calls-to-action, highlights, and key interactive elements to draw attention.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired appearance, suitable for headings and short, impactful statements.
- Body font: 'Inter' (sans-serif) to ensure excellent readability for longer text blocks, aligning with the clean and objective aesthetic.
- Utilize minimalistic line-art style icons throughout the website to complement the clean layout and maintain a premium, understated visual identity.
- A spacious, grid-based layout with ample negative space to enhance clarity, emphasize product visuals, and ensure all content is easily digestible and scannable.
- Implement subtle animations using Framer Motion, limited to gentle fade-ins for new content sections and responsive hover effects for interactive elements, ensuring a smooth yet unobtrusive user experience.
